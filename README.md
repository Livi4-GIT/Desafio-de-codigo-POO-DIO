# RELATÓRIO DE IMPLEMENTAÇÃO DE SERVIÇOS AWS

Data: 11/03/2026

Responsável: Livia Lirio

## Introdução
Este relatório apresenta o processo de implementação de ferramentas na empresa Abstergo Industries, realizado por Livia Lirio. O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos.

## Descrição do Projeto
O projeto de implementação de ferramentas foi dividido em três etapas, cada uma com objetivos específicos. A seguir, são descritas as etapas propostas para auxiliar na redução de custos da empresa com serviços em nuvem e apoiar a transição para uma distribuidora de produtos.

Etapa 1: 
- Ferramenta: Amazon S3
- Função: Armazenamento de dados e arquivos.
- A ferramenta Amazon S3 será de grande utilidade para a empresa, pois uma distribuidora lida com grande volume de dados, como notas fiscais, relatórios, backups de segurança, imagens de produtos e outros documentos importantes.

Com o S3 (Simple Storage Service), todos esses arquivos podem ser armazenados em um único ambiente seguro e escalável. Isso reduz custos com servidores físicos ou infraestrutura própria de armazenamento.

Além disso, a plataforma utiliza o modelo pay-as-you-go, no qual a empresa paga apenas pelo armazenamento realmente utilizado. Caso o volume de dados aumente, o serviço permite escalabilidade e a utilização de classes de armazenamento mais econômicas para arquivos antigos, reduzindo ainda mais os custos operacionais.

Etapa 2: 
- Ferramenta: AWS Lambda
- Função: Automação de processos.
- A ferramenta AWS Lambda pode apoiar diversos processos operacionais da distribuidora, como processamento de pedidos, geração de relatórios e integração com o sistema de estoque.

Com o Lambda, é possível executar códigos sem a necessidade de manter servidores físicos ou máquinas virtuais em funcionamento constante. O serviço segue um modelo serverless, no qual a execução ocorre apenas quando necessário.

Isso significa que o sistema não permanece ativo 24 horas, sendo cobrado apenas quando o código é executado. Dessa forma, além da redução de custos com infraestrutura, também é possível obter maior agilidade no processamento de pedidos, desde o momento em que o cliente realiza a compra até a atualização automática do estoque.

Etapa 3: 
- Ferramenta: AWS RDS
- Função: Gerenciamento de banco de dados.
- O serviço AWS RDS será utilizado para o armazenamento e gerenciamento de dados da empresa, como informações cadastrais de clientes, controle de estoque, pedidos e dados logísticos da distribuidora.

Com a utilização do Amazon RDS (Relational Database Service), a empresa reduz a necessidade de manutenção manual de servidores de banco de dados. O serviço oferece recursos automáticos como backup, atualizações e escalabilidade, permitindo que o banco de dados cresça conforme a necessidade da aplicação.

Dessa forma, a empresa reduz custos operacionais relacionados à administração de infraestrutura, garantindo ao mesmo tempo maior segurança, disponibilidade e confiabilidade dos dados.



## Conclusão
A implementação de ferramentas na empresa Abstergo Industries tem como esperado proporcionar maior eficiência operacional, redução de custos e melhoria no gerenciamento de dados e processos, o que amentaria a eficiência e a produtividade da empresa. Recomenda-se a continuidade da utilizxação das ferramentas implementadas e a busca por novas tecnologias que possam melhorar ainda mais os processos da empresa.
